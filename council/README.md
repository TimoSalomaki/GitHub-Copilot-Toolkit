# AI Council

This folder stores session files for the AI Council multi-model deliberation system.

For documentation on how the council works, see [docs/ai-council.md](../docs/ai-council.md).

## Session Files

Session files (`council-session-YYYY-MM-DD-HHMMSS.md`) contain the complete deliberation record for each council discussion, including contributions from all models, proposals, and votes.
